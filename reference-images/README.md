# Reference Images

本目录用于存放论文制图参考示例。AI 制作论文图片前，应先查看对应分类中的参考图，学习其版式、线条、配色、标题位置和整体风格。

目录分类：

- `架构图/`：系统总体架构、分层架构、模块关系图参考。
- `功能结构图/`：后台、移动端、AI 模块等功能结构图参考。
- `流程图/`：业务流程、用户流程、管理员流程参考。
- `用例图/`：参与者和用例关系参考。
- `ER图/`：实体之间关系、联系类型、基数标注等 E-R 图参考。
- `实体属性图/`：单个实体与字段属性展开图参考。
- `类图/`：详细设计类图参考。
- `时序图/`：对象交互、接口调用、模块协作顺序参考。

使用规则：

1. 参考图只用于学习风格和排版，不直接照抄与项目无关的内容。
2. 最终论文图必须以当前项目源码、数据库、页面或用户说明为依据。
3. 若参考图风格和学校格式要求冲突，以学校格式要求和用户要求为准。
4. 制作图片时优先打开对应分类中的 PNG 查看效果，需要复用风格或调整布局时再打开同名 `.drawio` 源文件。
